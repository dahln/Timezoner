# Timezoner

A simple .NET library to provide a list of time zones with Id, Name, and Offset information.

## Installation

```bash
dotnet add package Timezoner
```

## Features

- ✅ Simple and lightweight
- ✅ Provides timezone Id, Name, and Offset
- ✅ Built for .NET 10.0
- ✅ Zero dependencies
